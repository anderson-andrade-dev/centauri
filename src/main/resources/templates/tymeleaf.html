Apostila de Thymeleaf 3.1.2

Essa apostila visa ensinar o uso básico e avançado do Thymeleaf 3.1.2, um template engine moderno para Java. Aprenderemos como integrá-lo em aplicações Spring Boot e outros frameworks. Exemplos de código serão fornecidos ao longo do conteúdo.

1. Introdução ao Thymeleaf

Thymeleaf é um mecanismo de templates para Java que permite processar e gerar templates HTML, XML, entre outros, de forma dinâmica. É amplamente utilizado com o Spring Boot para criar aplicações web dinâmicas.

1.1. Configuração

Para começar a usar Thymeleaf, adicione a seguinte dependência em seu arquivo pom.xml se estiver usando Maven:

<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-thymeleaf</artifactId>
</dependency>

2. Sintaxe Básica

Thymeleaf usa uma sintaxe de atributos especiais em tags HTML, onde os prefixos como th: são usados para definir o comportamento.

2.1. Exibindo Variáveis

Para exibir uma variável no HTML, usamos o atributo th:text. Exemplo:

<p th:text="${mensagem}">Texto padrão</p>

Esse código renderiza o valor da variável mensagem no lugar de "Texto padrão".

2.2. Operadores do Thymeleaf

Thymeleaf oferece vários operadores para manipular dados dentro dos templates. Alguns deles incluem:

  • ${...}: Acessa variáveis no modelo.
  • *{...}: Acessa propriedades de formulários.
  • #{...}: Acessa mensagens i18n (internacionalização).

3. Estruturas de Controle

Thymeleaf permite o uso de estruturas condicionais e loops.

3.1. Condicionais

Para mostrar conteúdo condicionalmente, use th:if e th:unless. Exemplo:

<p th:if="${usuario != null}">Bem-vindo, <span th:text="${usuario.nome}"></span></p>

3.2. Iterações (Loops)

O atributo th:each permite iteração sobre listas. Exemplo:

<ul>
    <li th:each="produto : ${produtos}" th:text="${produto.nome}"></li>
</ul>

4. Trabalhando com Fragmentos

Os fragmentos em Thymeleaf permitem reutilização de código HTML em diferentes partes da aplicação.

4.1. Definindo um Fragmento

<div th:fragment="cabecalho">
    <h1>Bem-vindo ao site!</h1>
</div>

4.2. Usando um Fragmento

<div th:insert="fragments :: cabecalho"></div>

5. Formulários

O Thymeleaf oferece suporte completo para formulários HTML e Spring. Abaixo está um exemplo simples de formulário com binding de dados:

5.1. Exemplo de Formulário

<form th:action="@{/enviar}" method="post" th:object="${usuario}">
    <input type="text" th:field="*{nome}" />
    <input type="email" th:field="*{email}" />
    <button type="submit">Enviar</button>
</form>

Este formulário está associado a um objeto usuario no back-end e envia os dados para a URL /enviar.

6. Internacionalização

Thymeleaf suporta internacionalização através do uso de arquivos de mensagens e o operador #{...}. Exemplo:

<p th:text="#{mensagem.bemvindo}">Bem-vindo ao sistema!</p>

7. Templates Layout Dialect

O layout dialect permite criar layouts de maneira mais organizada e reutilizável. Um exemplo de como incluir uma página dentro de um layout:

<html xmlns:layout="http://www.ultraq.net.nz/thymeleaf/layout">
<head>
    <title>Layout Principal</title>
</head>
<body>
    <div layout:fragment="conteudo">Conteúdo será injetado aqui</div>
</body>
</html>

Conclusão

Thymeleaf 3.1.2 é uma poderosa ferramenta para gerar conteúdo dinâmico em aplicações web Java. Com ele, você pode manipular dados de maneira eficiente, criar layouts reutilizáveis, e ainda suportar funcionalidades avançadas como internacionalização e formulários.
